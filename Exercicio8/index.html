Principais formatações de titulos

negrito / destaque

Nesta frase temos o destaque quando utilizamos o B fazendo com que o digitado fique na cor negrito. Deixando uma forma não semantica

Já quando utilizamos a tag Também aparece o negrito, porem de uma forma semantica.

ítalico / ênfase

para utilizarmo o italico basta acrescentar a tag i ou utilizando o atalho ctrl + shift + p e selecionando a tag emmet envelope com abreviatura. Deixando a frase nesse formato de forma não semantica.

A outra maneira de se utilizar o italico na forma semantica é acrescentando o tag EM de 'emfase' como citado agora.

texto marcado

Podemos colocar tambem um texto marcado utiizando a tag MARK, tambem podemos acrscentar um pouco de CSS utilizando o style dentro de uma tag para trocar sua cor.
